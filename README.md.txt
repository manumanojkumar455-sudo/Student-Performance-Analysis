# Student Performance Analysis Using Pandas

## Objective
Analyze student performance data and generate meaningful insights.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Load CSV dataset
- Data Cleaning
- Missing Value Analysis
- Filtering
- Grouping
- Visualization

## Insights
- Highest scorer identified
- Department-wise performance analyzed
- Attendance statistics generated

## How to Run

bash
pip install pandas matplotlib
python analysis.py
